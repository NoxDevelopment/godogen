# Nox Loom — CC0 / Permissive Asset Library

**~1.4 GB, ~200 packs on disk.** Every entry is **redistributable + AI-training-safe** for
commercial use — the opposite of paid bundles (ELV/Unity) which forbid AI-training or redistribution.

- **CC0** — public domain, no conditions (the bulk of this library).
- **CC-BY** — same freedoms + a credit line (files/folders tagged `_CC-BY`; credits in each folder's `ATTRIBUTION.txt`).
- **OFL / MIT / Apache** (fonts) — embed/redistribute/modify + AI-safe; keep the license notice.

Assets live under `godogen/pieces/asset-kits/<category>/` (git-ignored — fetched on demand; this catalog is the tracked index). Re-fetch via the source URLs below.
