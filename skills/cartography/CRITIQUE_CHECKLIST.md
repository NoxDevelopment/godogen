# Cartography Critique Checklist — the map acceptance gate

This is the **map-specific** rubric layered on top of the general
[visual-judge](../visual-judge/SKILL.md) gate. Run visual-judge's **Five
Anti-Self-Deception Rules first** (look with your own eyes · ≥3 seeds · side-by-side
same-scale vs the named competitor · real context/live editor · green ≠ good), then
score the map dimensions below. No map render earns "clears the bar," ships to the
showcase, or gets swapped into a template until it passes this across **≥3 seeds**.

## Name the bar before you judge
Region/world → **Wonderdraft**. Library depth → **Inkarnate**. Battlemap →
**Dungeondraft**. Procedural depth → **Azgaar**. Hand-inked idiom → **Tolkien /
Pauline Baynes**. Put ours next to the actual exemplar at the same size.

## Map rubric (score 0–3; any 0 = REJECT; default lower when unsure)

| # | Dimension | The failing tell (0) → the bar (3) |
|---|---|---|
| 1 | **Restraint / negative space** | Wall-to-wall stamp carpet, no open ground → most of the sheet breathes; features sparse + intentional. **(Our #1 historic failure — weight it heaviest.)** |
| 2 | **One coherent idiom** | Two styles fighting (soft brown lumps vs sharp grey peaks) → every symbol one language, palette, line weight. |
| 3 | **Believable geography** | Random scatter; rivers that split downhill or don't reach the sea; mountains everywhere → ranges on a spine, rivers high→sea, forests belt foothills, deserts in rain shadow. |
| 4 | **Biome variety** | One tan fill under all the bumps → differentiated plains/forest/marsh/desert/tundra by fill + texture + symbol set. |
| 5 | **Landmark / POI layer** | No points of interest (a heightmap) → castles, towers, ruins, ports, bridges, ships, sea-beasts, mines, temples anchor the story. |
| 6 | **Relief structure** | Per-cell scatter of identical peaks → ridgeline-walked ranges, tallest on crest, tapering to foothills, scale/spacing variance. |
| 7 | **Water & coast** | Flat gradient sea, hard coast → wave/hatch texture, coastal echo lines, figure-ground clarity, rivers/lakes that read. |
| 8 | **Labels as typography** | Overlapping symbols, one size, straight → hierarchy (world>region>city>feature), curved on features, collision-avoided, letterspaced regions (see REAL_CARTOGRAPHY). |
| 9 | **Sheet as one object** | N pasted sprites, mixed light → one light direction, pooled ground-shadow, global paper/aging pass fusing all layers. |
| 10 | **Decorative apparatus** | Bare or generic → style-appropriate cartouche, compass rose, scale bar, legend, border, tasteful marginalia. |
| 11 | **Robustness across seeds** | Looks OK on the lucky seed, falls apart on others → holds across ≥3 seeds. |
| 12 | **Symbol-library depth** | ~a dozen stamps → a real categorized library (120+ core, growing toward Inkarnate breadth). |

**SHIP** only when 1–11 are ≥2 on **every** seed viewed and #1, #2, #3 are ≥2
without exception. Otherwise **ITERATE** with the ranked gaps. #12 is a
program-level bar, not a per-render blocker, but log the gap.

## The P9 case (why this checklist exists)
The P9 "topographical" render was called "a real jump / clears the bar" and swapped
into the showcase on **one seed (88)**, framed by the generator's own write-up,
with green tests standing in for quality. Re-judged with the Five Rules against
Wonderdraft side-by-side, it scored: restraint **0** (wall-to-wall carpet), idiom
**1** (brown-lump vs grey-peak split), biome **0** (one tan fill), POI **0** (none),
robustness **untested** (one seed). Real verdict: **ITERATE**, not SHIP. That gap
between the claim and the truth is exactly what this gate + visual-judge close.
